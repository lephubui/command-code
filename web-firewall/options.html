üìö Rule Documentation
Supported Actions: block, allow, allowAllRequests
Resource Types: main_frame, sub_frame, script, image, xmlhttprequest, stylesheet, font, media, other
Condition Types: urlFilter (simple pattern), regexFilter (regex pattern), domains (domain list)

‚öñÔ∏è Balanced Mode: Blocks common web threats including malicious scripts, tracking requests, SQL injection attempts, and analytics collection. Provides solid protection without interfering with normal browsing.
üîí Paranoid Mode: Maximum security with all Balanced protections plus:

    • /admin/login - Generic admin login pages
    • /phpmyadmin - MySQL database management interface
  • Enhanced Script Protection: Stricter filtering of potentially malicious code execution attempts
