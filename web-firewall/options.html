Web Firewall — Settings

Add custom declarativeNetRequest rules in JSON (array). Each rule must include action and condition. Example:

[
  {
    "priority": 1,
    "action": { "type": "block" },
    "condition": { "regexFilter": "(?i)(/\.env)" , "resourceTypes": ["xmlhttprequest", "main_frame"] }
  }
]
